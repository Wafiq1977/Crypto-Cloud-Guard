==================================================
KONSEP APLIKASI
==================================================

Aplikasi bekerja seperti Google Drive namun khusus untuk file encryption.

Flow aplikasi:

1. User register dan login
2. User masuk ke dashboard pribadi
3. User upload file
4. User memilih algoritma enkripsi
5. User memilih format output file
6. User membuat key/password enkripsi
7. Sistem mengenkripsi file
8. File terenkripsi dengan format yang telah di set otomatis tersimpan di cloud storage aplikasi 
9. User dapat:
   - melihat daftar file
   - mendownload file encrypted sesuai format 
   - mendekripsi kembali file, dan file otomatis kembali seperti semula 
   - menghapus file
   - melihat histori aktivitas


Semua file harus terhubung dengan akun masing-masing user.

==================================================

Struktur:

- file encrypted disimpan di server
- metadata file disimpan di database

==================================================
FITUR FILE ENCRYPTION
==================================================

Saat upload file:

User harus:
1. memilih file
2. memilih algoritma
3. memilih format output
4. memasukkan encryption key

Contoh format output:
- .enc
- .cipher
- .locked
- .txt encrypted

Setelah encrypt:
- file otomatis tersimpan
- masuk ke halaman "My Files"
- dapat didownload kapan saja
